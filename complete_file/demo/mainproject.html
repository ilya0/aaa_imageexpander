Single Pole Banner

DESCRIPTION

Each Pole Banner set includes two 36" fiberglass rods set into two aluminum brackets that insert into the top and bottom of the banner's pockets. Also included are the four 36" metal straps to fasten the hardware to a pole. CALL FOR CUSTOM SIZES OR VOLUME PRICING. Size: 3'w x 8'h, 3'w x 8'h

Features:

  • Mounts to any Pole
  • Banner attaches by clamp
  • Easy Assembly
  • Pole Pockets sewn at bottom and top
  • Double Sided Option
